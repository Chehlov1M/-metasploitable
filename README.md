Задание 2. Режимы сканирования Nmap и анализ трафика

Для демонстрации различий в сетевом взаимодействии были проведены сканирования в следующих режимах:

    SYN scan (-sS) — полуоткрытое сканирование, наиболее распространённый метод.
    FIN scan (-sF) — сканирование с флагом FIN, используется для обхода простых правил фаерволов.
    Xmas scan (-sX) — пакет с флагами FIN, PSH, URG («зажжённый как рождественская ёлка»).
    UDP scan (-sU) — сканирование UDP‑портов, основанное на ответах ICMP.

Трафик перехватывался с помощью Wireshark на интерфейсе vboxnet0. Для каждого типа сканирования сохранён отдельный файл захвата (.pcapng) в папке captures/.
